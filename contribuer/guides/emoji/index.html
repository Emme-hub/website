Utiliser des émojis#

Web 2020 oblige, il est possible d'insérer des émojis directement dans le texte en Markdown.

On utilise l'extension dédiée de PyMdown et en particulier l'index Twemoji :

  • consulter les informations spécifiques sur cet index dans la doc de l'extension
  • consulter les codes des émojis depuis la table de correspondance unicode ↔ emojis

Syntaxe#

Elle est semblable à la plupart des outils de discussion : :emoji_code:.

Voici quelques exemples d'émojis : :beginner: :birthday: :boom: :candle: :earth_americas: :earth_africa: :earth_asia: :eyeglasses: :globe_with_meridians: :heart: :heart_eyes: :parachute: :pirate_flag: :pray: :smile: :smiley: :sweat_smile: :slight_smile: :unicorn: :woman_levitate:
